A aplicação permite o cadastro de usuários, exercícios e treinos personalizados, possibilitando a organização dos
exercícios em diferentes dias da semana. Além disso, o sistema registra sessões de treino realizadas, armazena 
o histórico de evolução do usuário e mantém registros detalhados dos exercícios executados.
Funcionalidades:
Cadastro de usuários.
Cadastro de exercícios.
Classificação dos exercícios por grupo muscular e tipo.
Criação de treinos semanais.
Associação de exercícios aos treinos.
Registro de sessões de treino.
Histórico de evolução do usuário.
