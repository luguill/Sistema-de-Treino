O sistema de gerenciamento de treinos é um projeto implementado em Java e utiliza banco de dados em MySql. Essa aplicação permite registrar usuários, exercícios e treinos personalizados, podendo organizar os exercícios em diferentes dias da semana. Além disso, registra as sessões de treino completadas, guarda o histórico de progresso do usuário e detalha os exercícios realizados.
